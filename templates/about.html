What is this?

A simple Markdown to HTML converter built with Flask and Bootstrap. Built using the official Markdown Guide conventions.


How does it work?

Click on the Editor button in the top right and start typing Markdown in the text box; the output will automatically be displayed on the right-hand side. When you're satisfied with your Markdown, just copy-paste it to wherever it needs to go!
